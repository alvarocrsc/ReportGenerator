## Descripción de Bibliotecas Principales

### JavaFX 21.0.6
- **Propósito**: Framework de interfaz gráfica
- **Componentes**: Botones, tablas, campos de texto, diálogos
- **Licencia**: GPL v2 con excepción Classpath (permite uso comercial)

### Apache PDFBox 3.0.6
- **Propósito**: Creación y manipulación de documentos PDF
- **Uso**: Genera los informes PDF con tablas y gráficos
- **Licencia**: Apache 2.0 (muy permisiva)

### JFreeChart 1.5.6
- **Propósito**: Creación de gráficos estadísticos
- **Uso**: Genera el gráfico circular de clientes por ciudad
- **Licencia**: LGPL (permite uso en aplicaciones propietarias)

## Información sobre Licencias

### Uso Comercial

Todas las bibliotecas utilizadas **permiten uso comercial** sin necesidad de liberar el código fuente de la aplicación.

### Resumen de Licencias

- **Apache 2.0**: Muy permisiva, permite uso comercial y modificación
- **MIT**: Extremadamente permisiva, mínimas restricciones
- **BSD 3-Clause**: Permisiva, requiere mantener aviso de copyright
- **LGPL**: Permite uso en software propietario si se enlaza como biblioteca
- **GPL v2 + Classpath Exception**: Permite uso en aplicaciones sin hacerlas GPL
